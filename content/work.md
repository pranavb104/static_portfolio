## Selected List of Projects

1. [Building a phygital retail experience for Maybach (China)](https://drive.google.com/file/d/1U_bslosQ8UG6kvTMyE1xngv9o3_2OIIh/view?usp=share_link)
_Working in a small design agency, we built an interactive experience and deployed it at a couple of dealerships for Mercedes Maybach. Here my role was to work together with technical artists, designers & PO's to define the customer experience & technology stack and eventually build the HW, source it and write code to bring it all together._

2. [Designing customer experience journey for JLR](https://miro.com/app/board/uXjVGZIVUFQ=/?share_link_id=856184180424)
_Helped define and deconstruct the customer experience for a particular feature which included user research, workshop facilitation and working with different stakeholders across the business._

4. [A workshop for students to explore movement with interactive tech](https://www.pranavbe.net/posts/post8/).
_We were invited to Denmark to organize an explorative workshop to teach students to interface the body with the computer._ 

4. [0-1 MVP for White Table Interactive](https://www.pranavbe.net/posts/post7/)
_My first job while in the Netherlands. I was responsible to help with redefining their existing White Table product. Through a lot of experiments, understanding pain points and working with customers actively using this product, I built an MVP portable solution._

6. [Open Source Plastic Scanner Project](https://plasticscanner.com/about/)
_I helped replicate their first open source prototype of the board, making sure the NIR LED's were consistent and able to record the wavelength of different plastics. This project is unfortunately paused at the moment._

7. Building screen prototypes for BUCKS and other research projects.
_While in JLR, I made screen prototypes using webapp tech for a number of research/discovery projects. Unfortunately a lot of this cannot be shown openly due to NDA._
